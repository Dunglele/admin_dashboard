# Shop Admin Dashboard (Django)

Giao diện quản trị dành cho cửa hàng.

## 🚀 Hướng dẫn chạy
1. Truy cập thư mục: `cd admin_dashboard`
2. Tạo môi trường ảo: `python -m venv venv`
3. Kích hoạt: `.\venv\Scripts\activate` (Windows)
4. Cài đặt thư viện: `pip install -r requirements.txt`
5. Chạy Dashboard: `python manage.py runserver 8080`

## ⚙️ Cấu hình
- Sửa file `.env` để thay đổi `API_BASE_URL` trỏ về Backend.
